Branding & Design Guidelines (v1)

PHILOSOPHY
- Hand-drawn pencil / graphite aesthetic
- Human, quiet, observational
- No marker, no paint, no gradients

COLORS
Dark:
- Background: #000000 to #0A0A0A
- Pencil light: #D8D8D8 to #E6E6E6
- Eyes: #1A1A1A to #222222

Light (Reverse):
- Background: #FFFFFF or #FAFAF8
- Pencil: #1A1A1A to #333333

USAGE RULES
- Never recolor
- Never smooth lines
- Never add shadows or outlines
- Icon = face only
- Wordmark separate from icon
